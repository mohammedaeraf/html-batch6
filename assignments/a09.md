### üéØ **Objective:**

To practice DOM manipulation in JavaScript by:

* Changing **text content**
* Modifying **styles dynamically**
* Updating an **image** on button click

---

### üß† **Instructions:**

1. Create an HTML page with a profile section (Name, Profession, and Image).
2. Add **two buttons**:

   * One to **change the profile information** (name, profession, and background color).
   * Another to **change the profile picture**.
3. Use **JavaScript functions** and **`document.getElementById()`** to modify these elements dynamically.

---

### üß± **Starter Code:**

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Profile Info - DOM Manipulation</title>
    <style>
      body {
        font-family: Arial, sans-serif;
        text-align: center;
        margin-top: 40px;
      }
      .profile-card {
        display: inline-block;
        padding: 20px;
        border-radius: 15px;
        box-shadow: 0 0 10px gray;
        background-color: #f2f2f2;
      }
      img {
        width: 200px;
        border-radius: 10px;
        margin-top: 15px;
      }
      button {
        margin: 10px;
        padding: 10px 20px;
        border: none;
        background-color: steelblue;
        color: white;
        border-radius: 5px;
        cursor: pointer;
      }
      button:hover {
        background-color: darkblue;
      }
    </style>

    <script>
      // Function to change text and style
      function updateProfile() {
        let name = document.getElementById("profileName");
        let job = document.getElementById("profileJob");
        let card = document.getElementById("profileCard");

        // Update content
        name.textContent = "Aisha Khan";
        job.textContent = "Full Stack Developer";

        // Change background and text style
        card.style.backgroundColor = "#e0f7fa";
        name.style.color = "darkblue";
        job.style.fontStyle = "italic";
      }

      // Function to change the image
      function changePhoto() {
        let photo = document.getElementById("profileImg");
        photo.src = "assets/img/developer.jpg";

        let caption = document.getElementById("imgCaption");
        caption.textContent = "Aisha at Work";
      }
    </script>
  </head>

  <body>
    <h1>JavaScript DOM Example - Profile Updater</h1>
    <div class="profile-card" id="profileCard">
      <h2 id="profileName">John Doe</h2>
      <h3 id="profileJob">Web Designer</h3>
      <img src="assets/img/designer.jpg" id="profileImg" alt="Profile Photo" />
      <p id="imgCaption">John at Studio</p>
    </div>

    <div>
      <button onclick="updateProfile()">Change Profile Info</button>
      <button onclick="changePhoto()">Change Profile Picture</button>
    </div>
  </body>
</html>
```
